My Portfolio Website

A fully responsive personal portfolio website built from scratch using HTML, CSS, and JavaScript. This project was built as Task 1 of my Web Development Internship at Future Interns.

🔗 Live Demo: https://kalyanabojja.github.io/FUTURE_FS_1/
📂 Repository: https://github.com/kalyanabojja/FUTURE_FS_1


✨ Features


🎨 Clean, modern, and responsive design (works on mobile, tablet & desktop)
🧑‍💻 About Me section
🛠️ Skills section
📁 Projects showcase
📬 Contact section / form
⚡ Smooth navigation and animations



🛠️ Built With

HTML5 – structure and content
CSS3 – styling and responsive layout
JavaScript – interactivity
